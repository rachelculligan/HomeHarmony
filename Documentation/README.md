# 🏡 HomeHarmony

HomeHarmony is a modern family home management app built with **SwiftUI** and **SwiftData**.

The goal is to simplify everyday family life by combining cleaning schedules, meal planning, grocery shopping, rewards, and family organization into one beautiful app.

---

## ✨ Features

### 🧹 Cleaning
- Daily & recurring tasks
- Cleaning templates
- Progress tracking
- XP & rewards

### 🍽 Meals
- Recipe library
- Weekly meal planner
- Interactive recipe picker
- Grocery list generation
- Pantry management (coming soon)

### 👨‍👩‍👧 Family
- Household profiles
- Rewards
- Achievements
- Statistics
- Leaderboards

### 📊 Dashboard
- Daily overview
- Family progress
- Today's meals
- Upcoming tasks
